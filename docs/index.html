How It Works

Three seconds from voice to text.

Hold the Key

Press your hotkey (Globe, Option, or any custom combo). Recording starts instantly.

Speak

Say what you want. The overlay shows live transcription with a waveform.

Release

Let go. Final transcription runs and text is pasted at your cursor position.

Two Engines

Pick the best fit for your language and hardware.

GigaAM v3

Default · Native MLX

Russian-optimized CTC model running natively via MLX on Apple Silicon. Supports real-time streaming transcription. Fastest option for Russian dictation.

Whisper

whisper.cpp · 99 Languages

OpenAI's Whisper via whisper.cpp with Metal acceleration. 9 model sizes from tiny (75 MB) to large (3 GB). Best for multilingual use.

Install

Download the DMG or build from source.

# Build from source
git clone https://github.com/misteral/open-wispr.git
cd open-wispr
bash build.sh
# → installs to /Applications/OpenWispr.app

Requires macOS 14+ (Sonoma) on Apple Silicon. Accessibility & Microphone permissions are prompted on first launch.
