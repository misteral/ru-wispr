# Core Beliefs

Agent-first operating principles for RuWispr.

1. **Everything on-device.** No audio or text ever leaves the machine. Privacy is non-negotiable.
2. **Repository is the system of record.** All decisions, plans, and context live in the repo.
3. **AGENTS.md is a map, not a manual.** Short table of contents, pointers to `docs/`.
4. **Legibility over cleverness.** Code must be understandable to a future agent run.
5. **Simple and minimal.** RuWispr does one thing well — voice to text. No bloat, no feature creep.
6. **Enforce invariants mechanically.** CI, shellcheck, and structural tests — not prose reminders.
7. **Agents implement, humans steer.** Maximum work delegated to agents, humans define direction and validate.
8. **Corrections are cheap; waiting is expensive.** Short-lived PRs, minimal blocking gates.
9. **User feels it first.** When dev convenience conflicts with UX quality — the user wins.
